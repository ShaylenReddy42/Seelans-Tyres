Ensure to set the following environment variables before continuing

-----

AddressServiceApi -- https://localhost:5013

	Localhost for the Address Microservice

AdminCredentials__Email [Can be fictitious]

	Used to Seed the Admin account to the database with the Administrator role

AdminCredentials__Password

	Password used to create the account

EmailCredentials__Email [Requires a real email account]

	Email used to Send emails

EmailCredentials__Password

	This is a generated app password
	On the email [gmail] account you choose to use, 
	two-factor authentication has to be enabled
	on your google account to show app passwords. 
	You would then generate
	a custom app password. Don't specify the
	platform and give it any name you want.
	Copy that as your password

OrderServiceApi -- https://localhost:5011

	Localhost for the Order Microservice

SeelansTyresCustomerContext -- e.g. Server=.;Initial Catalog=SeelansTyresCustomerDb;Trusted_Connection=true;MultipleActiveResultSets=true

	Connection String to the SQL Server Database that holds the customers

SeelansTyresAddressContext -- e.g. Server=.;Initial Catalog=SeelansTyresAddressDb;Trusted_Connection=true;MultipleActiveResultSets=true

	Connection String to the SQL Server Database for the Address Microservice database

SeelansTyresOrderContext -- e.g. Server=.;Initial Catalog=SeelansTyresOrderDb;Trusted_Connection=true;MultipleActiveResultSets=true

	Connection String to the SQL Server Database for the Order Microservice database

SeelansTyresTyresContext -- e.g. Server=.;Initial Catalog=SeelansTyresTyresDb;Trusted_Connection=true;MultipleActiveResultSets=true

	Connection String to the SQL Server Database for the Tyres Microservice database

Token__Issuer -- https://localhost:5010

	Party that issues the token

Token__Key [Must be 32 bits long]

	Used to create the security key for jwt bearer authentication

TyresServiceApi -- https://localhost:5012

	Localhost for the Tyres Microservice
